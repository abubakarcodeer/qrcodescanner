# QRCodeScanner 📷

QRCodeScanner is an Android application that scans both **QR codes and barcodes** quickly and displays the result in a **BottomSheet dialog** for easy interaction. The result screen provides options to **redirect to links** or **copy scanned text** instantly. The app also includes useful scanning tools such as **flashlight support**, **scan from gallery**, and adding **vibration and beep** after scan complete.

Additionally, the app supports **QR code generation**, allowing users to create a QR code by entering **text or a URL**. Generated QR codes can also be **shared easily** with others. The project uses **external libraries for scanning and QR generation** to provide fast and reliable performance.

## Features
- 📷 Scan **QR codes and barcodes**
- 📄 Result displayed in **BottomSheet dialog**
- 🔗 **Open links** or **copy scanned text**
- 🔦 **Flashlight support** while scanning
- 🖼 **Scan QR code from gallery**
- 🔔 Optional **vibration and beep feedback**
- 🧾 **Generate QR code** from text or URL
- 📤 **Share generated QR codes**

## Built With
- Kotlin
- Android SDK
- CameraX
- Google ML Kit Barcode Scanning
- ZXing (Zebra Crossing) for QR code Generate

## Requirements
- Android 7.0 (API 24) or higher
